# Orientações de manutenção

- Sempre que alterar arquivos principais do painel (como `index.html` ou `Code.gs`), atualize também `Index_2.html` para manter a versão HTML local em sincronia.
- Na dúvida, replique no `Index_2.html` qualquer mudança visual ou de lógica feita para a versão oficial.
- Use mensagens de commit claras indicando quando a versão local foi atualizada.
